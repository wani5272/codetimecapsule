NAME
       fgetline() - gets a line from a file

SYNOPSIS
       int fgetline(f,buf,length)
       FILE *f;
       char *buf;
       int length;

DESCRIPTION
       fgetline() reads up to length characters from the input file using
       getc() (q.v.). The transfer is also terminated by a newline, end of
       file, or read error. The newline character is stripped and the string
       is NULL ('\0') terminated.

RETURNS
       >=0    the number of bytes read.

       < 0    if an error occurred.

NOTES
       none
