FILECLOSE(3)		  Schily's LIBRARY FUNCTIONS		  FILECLOSE(3)

NAME
       fileclose() - closes a file

SYNOPSIS
       fileclose(f)
       FILE *f;

DESCRIPTION
       fileclose() closes the file f and flushes all buffers.

RETURNS
       0      on success

       EOF    on error none

NOTES
       none
