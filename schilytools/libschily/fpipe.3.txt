NAME
       fpipe() - sets up a two ended pipe

SYNOPSIS
       int fpipe(pp)
       FILE *pp[2];

DESCRIPTION
       fpipe() sets up a two ended pipe:

       pp[0]  is set to a FILE open for reading.

       pp[1]  is open for writing.

RETURNS
       NULL a pipe could not be created.  other completed successfully.

NOTES
       none
