NAME
       printf() - formats and prints to standard output

SYNOPSIS
       printf(format, arg1, ..., argn)
       char *format;

DESCRIPTION
       printf() formats its arguments using the routine format().  See
       format() for information about the options for formatting.

RETURNS
       none

SEE ALSO
       format(3), fprintf(3)

NOTES
       printf() buffers its output internally until it is finished then the
       buffer is flushed.
