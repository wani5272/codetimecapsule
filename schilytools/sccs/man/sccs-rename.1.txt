NAME
       sccs-rename - rename files in the project list for the next commit

SYNOPSIS
       sccs rename [options] source destination

DESCRIPTION
       The rename subcommand marks the file source to be renamed to
       destination with the next commit.

       This subcommand is a project extension that does not exist in historic
       sccs implementations.

OPTIONS
EXAMPLES
       To rename the file xec.c to cmd.c call:

	      sccs rename xec.c cmd.c

ENVIRONMENT VARIABLES
       See environ(5) for descriptions of the following environment variables
       that affect the execution of sccs-rename(1): LANG, LC_ALL, LC_CTYPE,
       LC_MESSAGES, and NLSPATH.


       SCCS_NO_HELP
	      If set, sccs rename will not automatically call help(1) with the
	      SCCS error code in order to print a more helpful error message.
	      Scripts that depend on the exact error messages of SCCS commands
	      should set the environment variable SCCS_NO_HELP and set
	      LC_ALL=C.
